关于weekly topics


1) 获取最近topics

URL：/yourdomainname/mosesenglish/api/weekly/latest.m

required parameters: 

木有

optional parameters: 

size: 你需要的数据大小。默认：10个



2）获取任一topic

URL：/yourdomainname/mosesenglish/api/weekly/any.m

required parameters: 

year: 那一年

optional parameters: 

木有

关于weekday topics


1）获取某一周的weekday topics

URL：/yourdomainname/mosesenglish/api/weekday/entries.m

required parameters: 

weeklyTopicId：某周的weekly topic id

optional parameters: 

木有



2）获取随机的weekday topics

URL：/yourdomainname/mosesenglish/api/weekday/random.m

required parameters:

木有 

optional parameters: 

size：数据大小； 默认5
